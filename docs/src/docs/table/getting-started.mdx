## Installation

Package depends on:

- [react](https://www.npmjs.com/package/react)
- [react-dom](https://www.npmjs.com/package/react-dom)
- [@mantine/core](https://www.npmjs.com/package/@mantine/core)
- [@table-library/react-table-library](https://www.npmjs.com/package/@table-library/react-table-library)
- [styled-components](https://www.npmjs.com/package/styled-components)

Install with npm:

```sh
npm install @mantine/core @mantine/table @table-library/react-table-library styled-components
```

Install with yarn:

```sh
yarn add @mantine/core @mantine/table @table-library/react-table-library styled-components
```
